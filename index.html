Parte 1 - Entendendo Comunicação entre Client x Server

História da Web

Nas primeiras aulas entendemos um pouco do contexto onde o HTML se insere. O Professor Diogo Mainardes abordou a história dos primeiros computadores, como surgiram e grandes nomes da computação.

O que são Clients

Seguindo as aulas, começamos a entender o que são Clients, principalmente ao que se refere a Clients no linguajar Web, como funcionam os navegadores, aplicações Web e dispositivos móveis, suas diferenças e como cada um se comporta em relação à web.

O que são Servers

Assim como os Clients, aprendemos tudo em relação a Servers, questões de Hardware e Softwares em servidores e suas diferenças para o Client, tipos de servidores e sobre hospedagem de sites.

Linguagens de Programação

Com o entendimento entre Client x Server, passamos adiante entendendo como essa comunicação se dá. As linguagens são importantes para fazer com que tudo funcione e elas têm os lados do Server e Client, entendendo como Server-side e Client-side. É importante mencionar que HTML não é uma linguagem de programação.

Parte 2 - Introdução ao HTML na Prática

Com o primeiro entendimento apresentado, passamos então a ver como o HTML se comporta na prática, suas ferramentas utilizadas como o Visual Studio Code, onde estou fazendo este trabalho neste momento, por exemplo. Como instalamos alguns plugins úteis e mexemos na ferramenta.

Estrutura Básica do HTML

Discutimos a estrutura básica de um documento HTML. Vimos como um arquivo HTML é organizado com elementos como <html>, <head>, <body>, e a importância de elementos como <!DOCTYPE html> para garantir a compatibilidade entre diferentes navegadores.

Falando sobre tags

Abordamos a variedade de tags disponíveis no HTML e como elas são usadas para estruturar o conteúdo. Vimos exemplos práticos de tags comuns como <div>, <span>, <header>, e a função de cada uma na criação de uma página web semântica e bem organizada.

Atributos das Tags

Exploramos os atributos que podem ser adicionados às tags HTML para definir suas propriedades e comportamentos. Vimos exemplos de atributos como id, class, src, href, e como utilizá-los para estilizar e funcionalizar os elementos HTML.

Textos

Focamos na formatação de textos usando tags HTML. Aprendemos sobre tags específicas para textos como <p>, <h1> até <h6>, <strong>, e <br>, e como essas tags ajudam a definir a hierarquia e o estilo do conteúdo textual em uma página web.

Listas ordenadas e não ordenadas

Estudamos como criar listas ordenadas e não ordenadas em HTML. Utilizamos as tags <ul> e <ol> para criar listas e a tag <li> para definir os itens dentro dessas listas. Vimos exemplos práticos de quando utilizar cada tipo de lista e como elas podem melhorar a organização visual do conteúdo.

"HTML is not a programming language. It is a markup language that defines the structure of your content. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear or act in a certain way." - Tim Berners-Lee, o inventor da World Wide Web
